# chroma jack

- プレイヤーは、画面に表示される色だけを頼りに、現在の色を白へ近づけていく。

- 山札から色カードを1枚ずつ引き、そのカードを現在の色へ加えるか、捨てるかを判断する。

- 色を加えた結果、赤・緑・青のいずれかの成分が上限を超えるとバーストとなる。

- 数値計算ではなく、人間の色彩感覚と直感によって判断するゲームとする。

## プロトタイプの目的

以下を検証する。

- 色だけを見てカードを取るか捨てるか判断する行為が楽しいか
- 白へ近づく過程に気持ちよさがあるか
- バーストの緊張感が成立するか
- 数値を表示しなくてもプレイヤーが判断できるか
- 何度も挑戦したくなるか
