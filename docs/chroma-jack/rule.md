# ゲームルール

Chroma Jackは、表示される色だけを頼りに目標色へ近づける、1人用のスコアアタックです。プレイ中はRGB値や目標までの距離を見せず、色彩感覚による判断を中心にします。

## ラウンドの流れ

ラウンドの初期色と目標色は選んだルールで決まります。山札から候補カードが公開され、プレイヤーは候補から1枚を使う、候補をまとめて捨てる、現在の色で止める、のいずれかを選びます。

カードを使うと効果が現在色へ即座に反映され、次の候補へ進みます。山札を使い切るか、プレイヤーが止めるか、許容されないバーストが起きるとラウンド終了です。

## バーストと得点

RGB加算では上限、CMY減算では下限を超えるとバーストします。通常はそのラウンドが終了して得点を失いますが、選択したルールによっては境界で止めて続行できます。

通常終了時は、確定した色がルールの目標色に近いほど高得点になります。バーストを許容するルールでは、続行できる代わりに得点上限が下がります。

## 山札の方針

基本の山札は、RGB加算なら赤・緑・青、CMY減算ならシアン・マゼンタ・イエローを主成分とするカードを均等に含めます。主成分以外にも弱い揺らぎを持たせ、系統の読みやすさと色の多様さを両立します。カードの具体的な生成範囲や枚数は、プレイテストに応じてコード側で調整します。

## 選択できるルール

- RGB Additive: 暗い色からRGBを加算し、白を目指す基本ルールです。
- CMY Subtractive: 明るい色からCMYに対応するRGB成分を減算し、黒を目指す基本ルールです。
- Clamp Challenge: 複数候補から選び、一部のバースト後も続行できる代わりに得点面の不利を受けます。
- Special Deck: 通常の色加算に加えて、色の状態を変える効果や判断を助ける特殊カードを試すためのルールです。
