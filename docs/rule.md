# ゲームルール

Chroma Jackは、表示される色だけを頼りに現在の色を白へ近づける、1人用のスコアアタックです。プレイ中はRGB値や白までの距離を見せず、色彩感覚による判断を中心にします。

## ラウンドの流れ

ラウンドは暗めの色から始まり、山札から候補カードが公開されます。プレイヤーは候補から1枚を使う、候補をまとめて捨てる、現在の色で止める、のいずれかを選びます。

カードを使うと効果が現在色へ即座に反映され、次の候補へ進みます。山札を使い切るか、プレイヤーが止めるか、許容されないバーストが起きるとラウンド終了です。

## バーストと得点

色成分を加えすぎて上限を超えるとバーストします。通常はそのラウンドが終了して得点を失いますが、選択したルールによっては超過成分を上限へ固定して続行できます。

通常終了時は、確定した色が白に近いほど高得点になります。バーストを許容するルールでは、続行できる代わりに得点上限が下がります。

## 山札の方針

基本の山札は、赤・緑・青のどれかを主成分とするカードを均等に含めます。ほかの成分にも弱い揺らぎを加え、狙う色を読み取れることと混色の多様さを両立します。カードの具体的な生成範囲や枚数は、プレイテストに応じてコード側で調整します。

## 選択できるルール

- Classic: 色だけを見て1枚ずつ判断し、バーストの緊張感を確かめる基本ルールです。
- Clamp Challenge: 複数候補から選び、一部のバースト後も続行できる代わりに得点面の不利を受けます。
- Special Deck: 通常の色加算に加えて、色の状態を変える効果や判断を助ける特殊カードを試すためのルールです。
